    Experience

    项目一：幔安顿电商平台
    软件架构： JetBrains WebStorm 10.0.3 HTML5布局 + jQuery v1.7.2 + 原生JS
    主用：less sublime
    开发周期： 2015.7-2015.11
    项目描述：
    该电商平台为PC端项目，为了借助电子商务的全新手段获得全球竞争优势，节省客户与企业的时间和空间，提高交易效率为忙碌的上班族节省宝贵时间，通过足不出户的网络渠道，了解本公司产品信息，为公司销售业绩提升一个台阶,
    1、后台管理系统：管理商品、订单、类目、商品规格属性、用户管理以及内容发布等功能；
    2、会员系统：用户可以在该系统中查询已下的订单、收藏的商品、我的优惠卷、团购等信息；
    3、订单系统：提供下单、查询订单、修改订单状态、定时处理订单；
    4、搜索系统：提供商品的搜索功能；
    5、单点登录系统：为多个系统之间提供用户登录凭证一级查询登陆用户的信息；
    责任描述：
    1.使用h5、css3搭建页面结构，使用less编译样式；
    2.采用前端开发中整个框架的搭建；
    3.网站性能的优化与兼容处理，基本都是ie的兼容；
    4.采用Ajax技术与后台进行数据交互，并用模板引擎进行页面渲染；
    5.代码的整合、优化、压缩与合并；
    项目二：爱途旅游网（手机端）
    项目描述：
    近年来，国人的生活品质提升，渐渐注重休闲生活，旅游变成了热门话题，由于网络方便，快捷的特性，成为我们寻求旅游信息的最佳利器，由于上网订购机票及参加旅游团的的费用低廉，所以受到旅游人士的欢迎，针对旅游的特色，量身定制面向广大游客，内容丰富，功能强大的网站，让浏览者更方便更透彻的了解本地旅游景点，吸引更多的旅游者；
    1、会员模块： 实现的主要功能包括新用户注册、老用户登录，同时，管理员对用户进行管理；
    2、信息检索模块：该模块主要是方便用户对本站景点和酒店进行相关检索，并返回检索结果。用户可以根据自己不同的需求，系统根据用户选择的不同组合检索方式和输入的关键字进行相关检索，得到检索的信息；
    3、在线留言模块：方便用户和管理人员沟通，用户可以发表自己的看法也可以浏览其他用户的留言；
    4、信息浏览模块：实现的主要功能包括查看站内公告（最新公告及近期的所有公共）；
    责任描述：
    1. 使用rem以及媒体查询实现移动端页面布局，让不同的移动设备都能访问到同效果页面；
    2. 采用CSS3技术实现制作强交互、多动画效果页面；
    3. 完成登录页，注册页，密码找回页面的用户提议以及与后台数据的交换；
    4. 通过CSS3动画 + Javascript + 手势，增强页面和用户间的互动；
    5. 面对手机端品牌复杂，浏览器种类繁多的现状，处理了非常棘手的手机端浏览器兼容性问题；
